# Agentic AI System for Clinical Lab Communication

## Overview

This project implements an agentic AI workflow that analyzes structured laboratory results and generates draft clinician-to-patient communications. The system combines rule-based clinical logic with Large Language Model (LLM) inference to create understandable patient-facing explanations while maintaining safety and auditability.

## Features

- Analyzes laboratory test values and identifies abnormal results
- Assigns severity categories (Routine, Follow-Up Recommended, Urgent Follow-Up)
- Generates patient-friendly explanations using an LLM
- Applies safety validation to reduce hallucinations and unsupported medical recommendations
- Stores structured outputs for auditing and review
- Implements conditional decision-making and error handling throughout the workflow

## Technologies

- Python
- Large Language Models (LLMs)
- JSON
- Prompt Engineering
- Rule-Based Systems
- NLP

## Workflow

1. Analyze laboratory values
2. Identify abnormal findings
3. Assign severity level
4. Generate draft patient message
5. Perform safety validation
6. Save structured output
