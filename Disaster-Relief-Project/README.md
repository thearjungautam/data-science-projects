# Disaster Relief Prediction Using Aerial Imagery

## Overview

This project uses machine learning techniques to identify displaced populations following the 2010 Haiti earthquake. Using aerial imagery data, the models classify locations likely to contain temporary shelters and assist in prioritizing disaster relief efforts.

## Features

- Geospatial data preprocessing and feature engineering
- Classification of shelter and non-shelter locations
- Comparison of multiple machine learning algorithms
- Model evaluation using ROC-AUC, accuracy, sensitivity, and specificity
- Threshold optimization to balance false positives and false negatives

## Models Evaluated

- Logistic Regression
- Linear Discriminant Analysis (LDA)
- Quadratic Discriminant Analysis (QDA)
- K-Nearest Neighbors (KNN)
- Elastic Net
- Random Forest
- Support Vector Machine (SVM)

## Technologies

- R
- tidymodels
- Machine Learning
- Statistical Modeling
- Geospatial Data Analysis

## Results

The final models achieved over **85% classification accuracy** in identifying likely shelter zones within hold-out datasets, demonstrating the potential of machine learning to support humanitarian response efforts.

## Key Learning Outcomes

- Feature engineering
- Classification modeling
- Cross-validation
- Model selection and comparison
- Threshold optimization
- Disaster response analytics

## Applications

This project demonstrates how machine learning can support resource allocation and decision-making in disaster relief and humanitarian operations.
